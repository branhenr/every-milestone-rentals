# Every Milestone Rentals

A full-stack rental management web application built for a small business that rents tables and chairs for corporate events and galas.

## 🚀 Live Demo
*Coming soon after AWS deployment*

## 📋 Project Overview
Replaces manual Google Sheets + calendar workflows with a purpose-built app featuring:
- Public catalog for customers to browse inventory
- Customer inquiry form
- Admin dashboard for bookings, invoicing, and analytics
- Automated invoice generation on job completion
- Business profit and expense analytics

## 🛠️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Backend | Java 25, Spring Boot 3.x, Spring Security + JWT |
| Database | PostgreSQL / AWS RDS |
| Frontend | React 18, Vite, Tailwind CSS, Axios |
| Deployment | AWS Elastic Beanstalk, S3, CloudFront |
| CI/CD | GitHub Actions |
